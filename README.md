# 캡컷 에이전트

한국어 토킹 영상을 넣으면 **무음·잔말을 잘라낸 캡컷(CapCut) 드래프트**를 만들어 줍니다.
자막(세그먼트 단위)도 함께 넣어 주고, 마지막에 직접 보면서 다듬을 수 있습니다.

> **AI 자동 / 사람 손 경계** — 자르는 건 AI가 제안하고, 최종 확정은 사람이 합니다.

## 무엇을 해주나

1. **무음 제거** — 말 사이 빈 곳을 잘라 점프컷
2. **음성 인식** — 한국어 대본 추출(애플 실리콘 mlx-whisper)
3. **잔말 컷** — "음/어" 같은 군말 자동 제거 / 같은 말 반복(NG)은 "의심"으로 표시만
4. **자막** — 세그먼트 단위 자막 트랙
5. **검토 화면** — 원본을 보며 `[`/`]` 키로 보존 구간을 직접 다듬고, "쉼 감도"로 얼마나 잘릴지 조절
6. 결과는 **내 맥의 캡컷에 드래프트로 생성** → 캡컷에서 열어 마무리 편집

영상은 **내 컴퓨터 밖으로 나가지 않습니다**(로컬 전용).

## 필요 조건

- **애플 실리콘(M1/M2/M3…) 맥** — 음성 인식 엔진이 애플 실리콘 전용입니다. 인텔 맥·윈도우는 미지원.
- **캡컷(CapCut) 데스크톱 앱** 설치 — 결과 드래프트를 여기서 엽니다.
- **Python 3.10+**, **Homebrew**(ffmpeg 설치용), 디스크 여유 약 2GB(음성 모델).

## 설치

```bash
git clone https://github.com/dawon1982/capcutai.git
cd capcutai
```

그다음 **`install.command` 더블클릭** (또는 터미널에서 `./install.command`).
ffmpeg 설치 → 가상환경 → 패키지 → 음성 모델(약 1.5GB, 처음 한 번) 다운로드까지 자동입니다.

### ⚠️ "확인할 수 없습니다" / "악성 코드" 경고가 뜨면 (ZIP으로 받은 경우)

GitHub에서 **ZIP으로 다운로드**하면 macOS가 모든 파일에 "다운로드됨" 격리 딱지를 붙입니다.
서명되지 않은 스크립트라 차단되는데(최신 macOS는 우클릭 → 열기로도 안 됨), 아래 한 번이면 해결됩니다.

**방법 1 — 터미널 한 줄 (가장 확실, 권장)**
1. **Spotlight**(⌘+Space) → `터미널` 입력 → 열기
2. 터미널에 이걸 입력하고 **맨 끝에 한 칸 띄우기** (아직 Enter 누르지 말 것):
   ```
   xattr -dr com.apple.quarantine 
   ```
3. **압축 푼 `capcutai` 폴더를 터미널 창으로 드래그** → 경로가 자동으로 붙습니다 → **Enter**
4. 이제 `install.command` 더블클릭하면 정상 실행됩니다.

**방법 2 — git clone (경고 자체가 안 생김)**
ZIP 대신 터미널에서 `git clone https://github.com/dawon1982/capcutai.git` 로 받으면 격리 딱지가 안 붙어 바로 됩니다.

> 한 번만 풀어주면 `install.command`·`run.command` 둘 다 이후로는 그냥 더블클릭됩니다.

## 사용

**`run.command` 더블클릭** → 브라우저가 `http://localhost:8770` 으로 열립니다.

> **바로 체험**: 처음이라면 저장소에 들어 있는 `samples/demo_ko.mp4`(16초 한국어 데모)를
> 올려 보세요. 무음·군말 제거 + 자막이 어떻게 되는지 바로 확인할 수 있습니다.

1. 영상을 드래그(또는 클릭해서 선택)
2. 처리가 끝나면 검토 화면에서:
   - **space** 재생/정지 · **← →** 0.1초 이동(shift=프레임)
   - **`[`** 시작 · **`]`** 끝 = 보존 구간 추가 · **delete** 재생헤드 구간 삭제
   - **NG 막대 클릭** = 그 구간(재촬영 테이크) 제거
   - **"이보다 긴 쉼만 자르기"** 슬라이더로 컷 감도 조절
3. **드래프트 만들기** 클릭
4. **캡컷을 완전히 종료(⌘Q) 후 다시 열면** 목록에 새 드래프트가 보입니다. 열어서 재생/마무리.

## 알아둘 점

- 음성 인식 모델은 처음 한 번만 받고 그 뒤로는 캐시됩니다. 같은 영상은 다시 인식하지 않습니다.
